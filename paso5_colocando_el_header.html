En este paso veremos cómo colocar el header o cabecera en nuestro diseño.

El header puede tener las medidas que queramos. Yo en este caso he diseñado uno de 1350 píxels de ancho aproximadamente para que supere un poco el ancho que ocupa el área de las columnas, que es un poco menor de 1300 píxels.



Ahora vamos a colocarlo en el área libre por encima de las columnas.

A continuación, vamos a copiar la selección que acabamos de hacer pulsando CTRL + C (CMD + C si estamos usando MAC OS).
Ahora nos vamos al lienzo donde tenemos nuestro diseño, seleccionamos nuestro grupo de capas llamado "Header" y pulsamos CTRL + V (CMD + V)
